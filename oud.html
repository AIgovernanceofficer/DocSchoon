DocSchoon

Verwijder automatisch persoonsgegevens uit Word-, PowerPoint-, Excel- en tekstbestanden — lokaal, privé en met behoud van opmaak.

🇳🇱 Nederlands & Engels 🔒 Geen cloud, geen API ✅ AVG-bewust 🆓 Open source (MIT)
⬇️ Bekijk op GitHub 📖 Documentatie
Wat doet het

Schone documenten in één stap

📄

Alle gangbare formaten

Word (.docx), PowerPoint (.pptx), Excel (.xlsx) en platte tekst — inclusief tabellen, kop- en voetteksten en slide-notities.

🎨

Opmaak blijft intact

Lettertype, vet, kleur en lay-out blijven onveranderd. Alleen de tekst van gevonden persoonsgegevens wordt vervangen.

🇳🇱

Nederlands als standaard

Inclusief BSN-validatie met elfproef, postcodeherkenning en Nederlandse telefoonnummers. Uitbreidbaar naar Engels.

🏠

Volledig lokaal

Draait op je eigen machine. Bestanden verlaten je computer niet. Geen account, geen cloud, geen datadeling.

Aan de slag

Installatie in vier stappen

1

Repository ophalen

git clone https://github.com/AIgovernanceofficer/DocSchoon.git
cd DocSchoon
2

Virtuele omgeving aanmaken

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
3

Afhankelijkheden installeren

pip install -r requirements.txt
4

Taalmodel downloaden (groot, ~500 MB)

python -m spacy download nl_core_news_lg

Zonder dit model werkt de tool in beperkte modus: e-mail, BSN, telefoon en IBAN worden wél gevonden, namen en locaties nog niet.

Start daarna de webinterface:

streamlit run app.py

⚠️ Geen officiële privacy-applicatie

  • Dit is een experimenteel hulpmiddel, geen gecertificeerde of officieel goedgekeurde privacytool. De tool is niet gevalideerd door een toezichthouder en biedt geen juridische garanties.
  • Controleer het resultaat altijd handmatig — geen enkel model haalt 100% van de gegevens.
  • Het resultaat is juridisch doorgaans gepseudonimiseerd, niet volledig anoniem. Combinaties van resterende gegevens kunnen iemand alsnog identificeerbaar maken.
  • Raadpleeg bij twijfel over de AVG-status van verwerkingen je functionaris gegevensbescherming of juridisch adviseur.
